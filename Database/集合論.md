集合論

ここではリレーショナルデータベースの基礎理論としての「集合論」をまとめる。

# 集合

集合とは同じ属性を持つデータの集まり。
例えば「１桁の自然数」の集合は以下のとおりとなる。

``` X = {1,2,3,4,5,・・・} ```

X以外の集合は「補集合」という。

# 集合演算

## 和

記号は「U」
SQLのUNION（和）の頭文字「U」と覚えると覚えやすいかも。

``` X = {1,2,3} ```
``` Y = {2,3,4} ```
``` X U Y = {1,2,3,2,3,4} ```


## 差

記号は「－」

``` X = {1,2,3} ```
``` Y = {2,3,4} ```
``` X － Y = {1} ```

## 積

記号は「∩」

``` X = {1,2,3} ```
``` Y = {2,3,4} ```
``` X － Y = {2,3} ```

## 直積

記号は「×」

``` X = {1,2,3} ```
``` Y = {2,3,4} ```
``` X × Y = {(1,2),(1,3),(1,4),(2,2),(2,3),(2,4),(3,2),(3,3),(3,4)} ```
